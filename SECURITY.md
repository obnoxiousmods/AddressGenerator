# Security Policy

## Supported Scope

This repository intentionally supports:

- public extended keys
- public cryptocurrency addresses
- explorer-backed balance and transaction reporting

It intentionally does not support:

- recovery seed entry
- mnemonic parsing from user input
- wallet passphrase processing
- private key import

## Reporting

Open a private security report through GitHub Security Advisories if a change would weaken those boundaries or expose user-sensitive public data unexpectedly.
